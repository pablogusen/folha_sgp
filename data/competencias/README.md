# Versionamento de Dados por Competência

Esta pasta organiza os dados históricos por competência (ano-mês).

## Estrutura:
```
competencias/
├── 2025-11/
│   ├── holerites/          # PDFs originais desta competência
│   ├── resultado.json      # Dados processados
│   └── relatorio.html      # Relatório gerado
├── 2025-12/
└── 2026-01/
```

## Uso:
Ao processar uma nova competência, os arquivos são automaticamente organizados na pasta correspondente, mantendo histórico completo para análises temporais.
